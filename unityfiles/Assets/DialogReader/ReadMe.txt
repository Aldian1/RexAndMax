This text document acts as an instruction manual for setting up dialogue with the dialogue system

Poses are in groups of 6 for both main characters

1 = normal
2 = annoyed
3 - anxious
4 - shocked
5 - upset
6 - angry

there are currently 3 animations too choose from for dialogue

1 = none - no animation
2 = shake - shake on the spot
3 = bounce - bounce on the spot
4 = float - float into the scene

A normal dialogue sequence would be setup like so

[Character]
[Pose]
[Animation]
[Dialogue]
